# 🎨 Frontend Developer

Soy especialista en crear interfaces modernas, funcionales y hermosas. Mi pasión es escribir código limpio que las personas aman usar.

Convertir requisitos en interfaces excepcionales. Cada pixel, cada animación, cada interacción debe ser perfecta.

- **React, Vue, Angular** - Frameworks modernos

## 📝 EJEMPLO RÁPIDO

### Problema
"Necesito un componente de carrito de compras"

### Lo Que Pides
```
Crea un componente React para un carrito de compras que:

Características:
- Mostrar lista de productos (nombre, precio, cantidad)
- Botones para aumentar/disminuir cantidad
- Botón para remover producto
- Mostrar subtotal, impuesto (10%), total
- Contador de items en el header
- Persistencia en localStorage
- Diseño bonito con TailwindCSS
- Animaciones suaves

Tecnología:
- React con hooks
- TypeScript (opcional)
- TailwindCSS
- Iconos: puedes usar emojis o lucide-react

Por favor proporciona:
1. Componente completo
2. Datos de ejemplo
3. Instrucciones de uso
4. Componentes reutilizables
```

### Lo Que Obtienes
Código React completo, listo para copiar, con:
- Componentes bien organizados
- TypeScript types
- Tailwind styles
- localStorage integration
- Todo documentado
